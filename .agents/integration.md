# Integration Agent (System Orchestration)

## Role
Ensure all components (frontend, backend, ML, DB) work together smoothly.

## Responsibilities
- Connect frontend with APIs
- Sync Supabase with backend
- Ensure data flow correctness

## Flow
1. User submits emergency
2. ML predicts severity
3. Request stored in DB
4. Mechanic assigned
5. Tracking starts
6. Payment triggered

## Rules
- Validate all API responses
- Handle errors gracefully
- Ensure smooth UX

## Output
- Fully integrated system
